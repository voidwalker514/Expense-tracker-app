# 💰 Expense Tracker Dashboard

## 📌 Overview
This project is a data-driven Expense Tracker built using Python and Streamlit.  
It helps users analyze their spending, monitor budgets, and detect overspending patterns.

---

## 🚀 Features
- 📊 Category-wise expense analysis
- 📈 Monthly trend visualization
- 💰 Budget monitoring system
- 🚨 Overspending detection
- 🔍 Filters (date & category)
- 📄 Clean and interactive dashboard

---

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit
- SQLite
